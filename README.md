MyBrand – Work Showcase and Contact Portal

MyBrand is a modern, responsive personal portfolio website designed to showcase professional work, skills, and achievements. It also includes a contact portal that allows visitors to easily connect. The project is built using core web technologies — HTML, CSS, and JavaScript — with a focus on clean UI, responsiveness, and usability.

Features:
Personal branding and introduction section
Work showcase / projects section
Skills and technologies display
Fully responsive design for desktop, tablet, and mobile
Interactive contact form with client-side validation
Clean and modern UI using CSS styling
Lightweight and fast-loading web pages

Technologies Used:
HTML5 – Structure and content of the website  
CSS3 – Styling, layout, and responsiveness  
JavaScript (Vanilla JS) – Interactivity and form validation
